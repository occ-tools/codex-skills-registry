SCHEMA_VALIDATION_FAILED Schema validation failed

A registry document does not match the published Codex Skills schema.

Fix: Fix the reported field, then rerun codex-skills validate or doctor.

CONFIG_SKILLS_MISSING Configured skills list is missing

A skill config file was provided but did not include a skills array.

Fix: Add a skills array or remove the invalid config file.

CONFIG_SKILL_INVALID Configured skill is invalid

A skill entry loaded from config could not be validated as a Codex Skill.

Fix: Fix the configured skill metadata or remove the invalid entry.

SKILL_FILE_MISSING Missing SKILL.md file

A discovered skill directory does not contain a readable SKILL.md file.

Fix: Add SKILL.md or remove the directory from the registry scan path.

SKILL_MARKDOWN_INVALID Invalid skill Markdown

A SKILL.md file could not be parsed into valid frontmatter and content.

Fix: Fix the frontmatter syntax and required skill metadata fields.

SKILL_DISABLED Disabled skill skipped

A skill was marked disabled and skipped by discovery.

Fix: Enable the skill only after it is ready for validation and use.

SKILL_DUPLICATE Duplicate skill name

Two discovered skills use the same registry name.

Fix: Rename one skill or remove the duplicate registration.

SKILL_NOT_REGISTERED Skill is not registered

A requested skill name does not exist in the loaded registry.

Fix: Use a registered skill name or add the missing skill to the project.

SKILL_NOT_ALLOWED Skill not allowed by policy

The project policy allows only specific skills and this skill is outside that list.

Fix: Add the skill to allowedSkills or remove it from the registry.

SKILL_DENIED Skill denied by policy

The project policy explicitly denies this skill.

Fix: Remove the skill or update deniedSkills after maintainer review.

SKILL_ABSOLUTE_ENTRY_POINT Absolute skill entry point

A skill entryPoint uses an absolute path instead of a skill-local path.

Fix: Use a relative entryPoint such as scripts/run.js.

MCP_SHELL_COMMAND Shell-based MCP command

An MCP server is launched through a shell command, which can expand arguments and execute arbitrary shell code.

Fix: Use a direct executable command and restrict enabled_tools or disabled_tools.

MCP_UNPINNED_NPX Unpinned npx package

An npx-based MCP server does not pin a full package version, so CI can run different code over time.

Fix: Pin the package argument with a full version such as @scope/server@1.2.3.

MCP_TOOL_POLICY_MISSING Missing MCP tool exposure policy

The MCP server does not declare enabled_tools or disabled_tools, making the exposed tool surface unclear.

Fix: Declare enabled_tools for least privilege, or disabled_tools for explicit exclusions.

MCP_CONFIG_PARSE_FAILED MCP config parse failed

A Codex MCP config file could not be parsed.

Fix: Fix the TOML/JSON syntax or remove the invalid MCP config file.

MCP_SERVER_NOT_ALLOWED MCP server not allowed by policy

The project policy allows only specific MCP servers and this server is outside that list.

Fix: Add the server to allowedMcpServers or remove the server config.

MCP_SERVER_DENIED MCP server denied by policy

The project policy explicitly denies this MCP server.

Fix: Remove the server or update deniedMcpServers after maintainer review.

MCP_COMMAND_NOT_ALLOWED MCP command not allowed by policy

The MCP server command is outside the policy allow-list.

Fix: Use an allowed command wrapper or update allowedMcpCommands after review.

MCP_COMMAND_DENIED MCP command denied by policy

The MCP server command is explicitly denied by project policy.

Fix: Replace the command or update deniedMcpCommands after maintainer review.

MCP_BROAD_APPROVAL_MODE Broad MCP approval mode

The MCP server uses an always or never default tool approval mode.

Fix: Prefer prompt-based or per-tool approval unless the broad mode is intentional.

MCP_INVALID_REMOTE_URL Invalid remote MCP URL

A remote MCP server URL is not a valid absolute URL.

Fix: Use a valid URL such as https://example.com/mcp.

MCP_REMOTE_NOT_HTTPS Remote MCP server without HTTPS

Remote MCP traffic over non-HTTPS transports can expose requests or credentials.

Fix: Use an HTTPS endpoint or add a documented suppression for an intentional local-only exception.

MCP_REMOTE_HOST_NOT_ALLOWED Remote MCP host not allowed by policy

The remote MCP endpoint host is outside the policy allow-list.

Fix: Use an approved host or add the host to allowedRemoteMcpHosts after review.

MCP_REMOTE_HOST_DENIED Remote MCP host denied by policy

The remote MCP endpoint host is explicitly denied by project policy.

Fix: Use a different endpoint or update deniedRemoteMcpHosts after review.

MCP_REMOTE_AUTH_UNDOCUMENTED Remote MCP auth is undocumented

Strict mode found a remote MCP server without bearer_token_env_var.

Fix: Set bearer_token_env_var or document why the endpoint does not require auth.

MCP_SECRET_LITERAL Potential secret literal

A value in MCP env or headers looks like a committed token or credential.

Fix: Move the secret to an environment variable and commit only the variable name.

MCP_REMOTE_URL_SECRET Credential-like MCP URL query

A remote MCP URL appears to include a token, key, secret, or credential value.

Fix: Move remote MCP credentials to bearer_token_env_var or env_http_headers.

MCP_REMOTE_TOKEN_ENV_VAR_INVALID Invalid MCP token environment variable name

bearer_token_env_var should name an environment variable, not use shell-specific or literal value syntax.

Fix: Use a portable variable name such as MCP_TOKEN.

MCP_HEADER_ENV_VAR_INVALID Invalid MCP header environment variable name

env_http_headers values should be environment variable names that resolve at runtime.

Fix: Use uppercase variable names such as MCP_AUTH_HEADER.

WORKFLOW_PERMISSIONS_MISSING Missing explicit workflow permissions

A GitHub Actions workflow does not explicitly declare GITHUB_TOKEN permissions.

Fix: Add least-privilege top-level or job-level permissions.

WORKFLOW_BROAD_PERMISSIONS Broad workflow token permissions

A workflow grants write-all or high-risk write permissions to GITHUB_TOKEN.

Fix: Scope write permissions to the smallest job and permission set required.

WORKFLOW_PULL_REQUEST_TARGET pull_request_target workflow

pull_request_target runs with elevated repository context and can expose write tokens to unsafe logic.

Fix: Prefer pull_request or isolate untrusted checkout, input interpolation, and write operations.

WORKFLOW_UNPINNED_ACTION Unpinned workflow action

A workflow action reference uses a mutable tag or branch instead of a full commit SHA.

Fix: Pin action references to audited 40-character commit SHAs.

WORKFLOW_UNTRUSTED_PR_INPUT Untrusted PR input in shell script

A pull request workflow interpolates PR event data directly into a run step.

Fix: Pass PR values through environment variables and quote shell references carefully.

WORKFLOW_DOWNLOAD_EXECUTE Downloaded script execution

A workflow pipes downloaded content into a shell.

Fix: Vendor the script, verify checksums, or use a pinned action.

SKILL_ENTRY_POINT_ESCAPE Skill entry point escapes its directory

The configured entryPoint resolves outside the skill directory.

Fix: Use a relative path inside the skill folder and avoid .. path segments.

SKILL_ENTRY_POINT_MISSING Skill entry point file is missing

A skill entryPoint references a file that was not found.

Fix: Create the entryPoint file or update the skill metadata to the correct path.

SKILL_INSTRUCTION_ONLY Instruction-only skill

Strict mode found a skill without an executable entryPoint.

Fix: Add an entryPoint or suppress the warning with a documented reason.

SKILL_SECURITY_TAG_MISSING Security skill tag missing

A skill triggered by security does not include a security tag.

Fix: Add the security tag so review and routing workflows can identify it.

PLUGIN_MCP_PATH_MISSING Plugin MCP path is missing

Fix: Create the referenced MCP config or update the plugin manifest path.

PLUGIN_MCP_PATH_ESCAPE Plugin MCP path escapes plugin root

PLUGIN_MCP_NOT_DISCOVERED Plugin MCP server not discovered
